## Usage

- Clone repo
- Add extension inside `chrome://extensions/`
- Press `Ctrl+Shift+F` to toggle custom sheet

## Extension

- Add sheet inside [sheets/](sheets/)
- Add if statement inside function [`getStyleSheet`](background.js)
